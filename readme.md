## Installation

### Git

```bash
git clone https://github.com/benja2998/git-ts
cd git-ts
npm ci
npm i
npm run build
npm link
```

Then you'll have `git-ts` in your PATH.

### NPM

```bash
npm i -g git-ts
```

Then you'll have `git-ts` in your PATH.
