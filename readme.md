## Installation

```bash
git clone https://github.com/benja2998/git-ts
cd git-ts
npm ci
npm i
npm run build
npm link
```

Then you'll have `git-ts` in your PATH.

## Neovim integration

See example-init.lua for information. Note that the plugin requires you to have installed git-ts and have it in your PATH.
